Para compartilhar imagens Docker utilizamos registries.

O mais comum é o **Docker Hub**.

---

## Login no Docker Hub

docker login

---

## Criando uma tag da imagem

docker tag getting-started SEU_USUARIO/getting-started

---

## Enviando a imagem

docker push SEU_USUARIO/getting-started
