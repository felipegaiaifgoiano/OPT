## O que é Docker?

Docker é uma plataforma que permite empacotar aplicações e suas dependências em **containers**.

Containers garantem que uma aplicação rode da mesma forma em diferentes ambientes, como:

- máquina local
- servidores
- ambientes de cloud

---

## O que é um Container?

Um container é um processo isolado que inclui:

- aplicação
- dependências
- bibliotecas
- configurações

Containers são leves e iniciam rapidamente.

---

## Como o Docker funciona

Normalmente o fluxo de trabalho com Docker segue estes passos:

1. Criar uma **imagem Docker**
2. Executar um **container** baseado nessa imagem
3. Interagir com a aplicação

Neste workshop você seguirá exatamente esse fluxo.
