# O que você vai aprender

Durante este tutorial você irá aprender:

- Executar aplicações em containers
- Criar imagens Docker
- Atualizar aplicações em execução
- Compartilhar imagens usando o Docker Hub
- Persistir dados com volumes
- Executar aplicações com múltiplos containers
- Utilizar Docker Compose

Esses conceitos formam a base para trabalhar com Docker em ambientes de desenvolvimento e produção.
